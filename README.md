# Flame
Static site generator for lua

## Requirments
   - Lua
   - Perl(used for markdown.pl)
   - cURL
   - GNU Coreutils
   - unzip

## Usage
```
flame - Static Site Generator
Usage: flame [command]
Commands:
  init          - Create new site
  build         - Build current site
  serve         - Serve current site
```

## Themes
Themes contain a ```header``` and a ```footer``` with a ```style```.
